# ddds

## 概要

dddsはDDDの開発補助ライブラリであり、型を使った制約により実装ミスをコンパイルエラー化する事を目的としています。
dddsという単語はdomain-driven-design-supportの略語です。  
エンティティやリポジトリ、イベント等、DDDの戦術的設計要素の実装を補助するインターフェースを提供します。

## 依存関係

dddsは永続的なデータ構造や関数を利用するために[vavr]に依存しています。
